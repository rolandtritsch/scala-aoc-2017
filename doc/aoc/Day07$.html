object Day07

Problem: https://adventofcode.com/2017/day/7

Solution:

General - This is a tree searching problem.

Part1 - The root is the only node with no parent, means it is never referenced as a child, means we can find it by doing the diff between all nodes and all children.

Part2 - Find the root. Go through the tree (top-down) and find the (bad) node, where the tree unbalanced (for the first time). Find the wrong weight (the one that is different to the other ones) and calc you to correct it.
