object Day09

Problem: https://adventofcode.com/2017/day/9

Solution:

General - My first implementation was based on a recursive processing of the input stream. Hard to read. Hard to understand. Hard to extend/ maintain. And hard to run (needs a special -Xss setting (more/large stack).

This (refactored) implementation is using a state machine. The states are OutOfGroup, InGroup, InGarbage, InCanceled (pic). While I am running the state machine I am collecting stats that will allow me (at the end) to answer (the given) questions about the input stream.

Part1 - Trivial. Collect and show the right stats.

Part2 - Trivial. Collect and show the right stats.
