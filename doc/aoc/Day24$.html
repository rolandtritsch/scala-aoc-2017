object Day24

Problem: http://adventofcode.com/2017/day/24

Solution:

General - Kind of simple. First you need to findZero. Then you need to findPath for one component. And then you need to findPaths for all components. You basically build a tree of all possible paths/solutions. And on that you can then look for the strongest and/or for the longest path and/or for ...

Part1 - Just look for the strongest path (but that might not be the longest path).

Part2 - Now look for the longest path (and if you find more than one longest one, pick the strongest of the long ones).
